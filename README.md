
# Проект Mesto
## Описание
Вёрстка интерактивной страницы, куда можно добавлять фотографии, удалять их и ставить лайки.
## Функциональность
На данный момент присутсвуют:
+ Базовая вёрстка сайта
+ Поп-ап (без функциональности)
## Используемые технологии
+ HTML
